# Настройка PWA

Приложение настроено как Progressive Web App (PWA) и может быть установлено на мобильных устройствах.

## Генерация иконок PNG

Для генерации PNG иконок используйте один из способов:

### Способ 1: HTML генератор (рекомендуется)
1. Откройте файл `public/generate-icons.html` в браузере
2. Иконки будут автоматически скачаны
3. Переместите файлы `icon-192.png` и `icon-512.png` в папку `public/`

### Способ 2: Онлайн конвертер
1. Используйте любой онлайн конвертер SVG в PNG (например, https://convertio.co/svg-png/)
2. Загрузите `public/icon.svg`
3. Создайте иконки размеров 192x192 и 512x512
4. Сохраните как `icon-192.png` и `icon-512.png` в папку `public/`

## Установка на устройство

### Android (Chrome):
1. Откройте приложение в браузере Chrome
2. Нажмите на меню (три точки)
3. Выберите "Добавить на главный экран" или "Установить приложение"

### iOS (Safari):
1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран «Домой»"

## Файлы PWA

- `public/manifest.json` - манифест приложения
- `public/service-worker.js` - service worker для кеширования
- `public/icon.svg` - SVG иконка (используется как fallback)
- `public/icon-192.png` - иконка 192x192 (нужно сгенерировать)
- `public/icon-512.png` - иконка 512x512 (нужно сгенерировать)

## Проверка PWA

После сборки проекта (`npm run build`), откройте `dist/index.html` в браузере и проверьте:
- В DevTools > Application > Manifest - должен быть виден манифест
- В DevTools > Application > Service Workers - должен быть зарегистрирован service worker

## Примечание

Приложение будет работать и без PNG иконок (используется SVG), но для лучшей совместимости рекомендуется создать PNG иконки.
